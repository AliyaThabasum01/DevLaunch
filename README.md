# 🚀 DevLaunch

A minimal developer dashboard built with HTML, CSS, and JavaScript.

## Features

- Live clock
- Dynamic greeting
- Random coding quotes
- Daily coding challenge
- Useful developer links
- Dark/Light mode toggle

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Installation

```bash
git clone https://github.com/yourusername/devlaunch.git
```

Open `index.html` in your browser.
